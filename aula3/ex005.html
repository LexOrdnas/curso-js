Operadores Aritméticos

        5 + 2 = 7 (Adição)
        5 - 2 = 3 (Subtração)
        5 * 2 = 10 (Multiplicação)
        5 / 2 = 2.5 (Divisão Real)
        5 % 2 = 1 (Resto da Divisão Inteira)
        5 ** 2 = 25 (Potenciação)

        -----------------------------------------
    
        Ordem de Precedência

        |  ()

        |  * / %
        |  + -
        |  > < >= <= == !=
        |  !
        |  &&

        -----------------------------------------

        Atribuição Simples

        var a = 5 + 3 (8)
        var b = a % 5 (3)
        var c = 5 * b ** 2 (45)
        var d = 10 - a / 2 (6)
        var e = 6 * 2 / d (2)
        var f = b % e + 4 / e (3)

        -----------------------------------------

        Auto-Atribuições || Simplificando

        var n = 3 
        n = n + 4      |      n += 4
        n = n - 5      |      n -= 5
        n = n * 4      |      n *= 4
        n = n / 2      |      n /= 2
        n = n ** 2     |      n **= 2
        n = n % 5      |      n %= 5

        -----------------------------------------

        Incremento

        var x = 5
        x = x + 1   |   x++  ou  ++x
        x = x - 1   |   x--  ou  --x
        -----------------------------------------

        typeof verifica o tipo de dado selecionado.

        -----------------------------------------

        Relacionais

        5 > 2 = true
        7 < 4 = false
        8 >= 8 = true
        9 <= 7 = false
        5 == 5 = true
        4 != 4 = false

        -----------------------------------------

        Identidade

        5 == 5 = true
        5 == '5' = true
        5 === '5' = false
        5 === 5 = true

        -----------------------------------------

        Lógicos

        ! negação (NOT)
        && conjunção (E)
        || disjunção (OU)

        -----------------------------------------

        Exemplos

        idade >= 15 && idade <= 17
        estado == 'RJ' || estado == 'SP'
        salario > 1500 && sexo != 'M'

        -----------------------------------------

        Ternário

        teste ? true : false

        ex.: media >= 7.0 ? "Aprovado : Reprovado"

        var x = 8

        var res = x % 2 == 0 ? 5 : 9

        var idade = 19

        var r = idade >= 18 ? 'MAIOR' : 'MENOR'
